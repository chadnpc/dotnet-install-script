# dotnet-install-script

A custom dotnet install script
# Install

```PowerShell
Install-Script -Name Install-Dotnet -Scope CurrentUser -Force
```

# usage

```PowerShell
Install-Dotnet -Channel LTS ...
```

# Uninstall

```PowerShell
Uninstall-Script -Name Install-Dotnet -Scope CurrentUser -Force
```
